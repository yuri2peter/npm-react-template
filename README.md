# React 组件开发模板

## 特性

- Typescript 支持
- 实时 React 组件开发编译
- 实时 React 子项目引用开发组件并预览

## 开发

1 - 安装依赖

```
> npm ci
```

2 - 实时 build

```
> npm start
```

3 - 另外开启一个终端窗口，前往 (sample)[`/sample`] 子项目，安装依赖并实时 build

```
> cd sample
> npm i
> npm run start
```

完成！现在你可以一边编写组件源码一边实时预览到它在项目中的效果。

## 发布

1. 发布前，请检查并替换`npm-react-template`关键字和相关项目信息为真实信息。
2. `> npm run release`
